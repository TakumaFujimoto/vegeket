## セクション2について
本セクションではプロジェクトを進めていくにための大まかな準備を行います。

仮想環境の作成もここで行います。

全体像を意識しながら、djangoのコマンドを使って基本的なファイルやフォルダを作成し、さらに基本的な設定もここで行っておきます。


## 仮想環境

powershell
```
python -m venv venv
```

### Windowsでの仮想環境のアクティブ化

venv\Scripts\activate
